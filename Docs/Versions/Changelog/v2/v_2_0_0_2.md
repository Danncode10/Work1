## Usage Instructions

### Setup Steps
1. Follow the checklist in `Docs/Versions/Version 2.md` v2_0_0_2 section
2. Use the obtained access_token in n8n workflows

### Configuration
- Temporarily set `NODE_ENV=development` for token acquisition
- Return to `NODE_ENV=production` for secure operation

### For n8n Integration
1. Complete the token acquisition process
2. In n8n HTTP Request node:
   - Authentication: Generic Auth
   - Auth Type: Header Auth
   - Header Name: `Authorization`
   - Header Value: `Bearer <your_access_token>`
