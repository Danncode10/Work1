## Notes
The DATABASE_URL `postgresql://localhost:5432/naturalhealthdb` follows PostgreSQL's standard connection string format:
- `postgresql://` = Protocol identifier
- `localhost` = Host (127.0.0.1 for local machine)
- `5432` = Default PostgreSQL port
- `naturalhealthdb` = Database name

For local Homebrew PostgreSQL, no username/password required (connects using current system user). In production environments, the URL includes credentials and remote host details.
